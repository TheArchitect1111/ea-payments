# Error monitoring setup (`ea-payments`)

**Canonical guide:** [GLITCHTIP-SETUP.md](./GLITCHTIP-SETUP.md)

Efficiency Architects uses **GlitchTip** (Sentry-compatible protocol) via `@sentry/nextjs`.

Preferred env var: `NEXT_PUBLIC_GLITCHTIP_DSN`.

Legacy `NEXT_PUBLIC_SENTRY_DSN` is still accepted during migration.
