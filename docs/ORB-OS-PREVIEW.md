# Orb OS Preview (legacy chat-first)

**Default product:** [SIMPLIFI Orb System](./SIMPLIFI-ORB-SYSTEM.md) — Brief home + corner Orb.

The conversation-first shell (`OrbOsShell`) is **experimental only**:

| URL | Behavior |
|-----|----------|
| `/simplifi/orb` | Redirects to `/simplifi/workspace` |
| `/simplifi/orb?chat=1` | Legacy centered chat shell |

Do not enable chat-first as the default home. Prefer the corner Orb expanded panel for Ask / voice.
