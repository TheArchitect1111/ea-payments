# Orb OS Preview (legacy chat-first)

**Default product:** [SIMPLIFI Orb System](./SIMPLIFI-ORB-SYSTEM.md) — Brief home + corner Orb.

The conversation-first shell (`OrbOsShell`) is **deprecated**. Prefer the corner Orb expanded panel for Ask / voice.

| URL | Behavior |
|-----|----------|
| `/simplifi/orb` | Redirects to `/simplifi/workspace` |
| `/simplifi/orb?chat=1` | Redirects to Brief (no longer opens chat shell) |
| `/simplifi/orb?chat=1&legacy=1` | Deprecated centered chat shell (escape hatch only) |

Do not enable chat-first as the default home.
